paragraph tag is a block level tag (means those elements which cover the entire row in which they are displayed. ). Mostly we use it to display some text on the page. paragraph tag automaticaly generate a new line at the begining as well as at the ending. Bydefault paragraph are aligned left.But we can also align it in center as well as in right by using css and html attribute align.
